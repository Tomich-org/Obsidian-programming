# Из чего состоит стартовая строка HTTP ответа?
**Версия протокола** (`HTTP/1.0`), **код состояния** (`200`), **пояснение** (`OK`)
# Какие бывают коды ответа?
- 200 - успех
- 300 - перенаправления, редиректы
- 400 - ошибки на стороне клиента (например неправильный запрос)
- 500 - ошибки со стороны сервера
- 504 - таймаут при проксировании (например прокси сервер передал запрос микросервису а тот оказался недоступен)
