В Python есть шесть операций сравнения и все они имеют одинаковый приоритет, который выше чем у [[Битовые операции]].

Список операций сравнения:
- `x < y` - строго `x` меньше `y`,
- `x <= y` - `x` меньше или равно `y`,
- `x > y` - строго `x` больше `y`,
- `x >= y` - `x` больше или равно `y`,
- `x == y` - `x` равно `y`,
- `x != y` - `x` не равно `y`.
